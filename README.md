# LinkedList

Script ini dibuat menggunakan java dan ditujukan untuk pemahaman dalam mata kuliah struktur data dan algoritma


How to use

```sh
$ git clone https://github.com/ghazimuharam/LinkedList/
$ cd LinkedList
```

To compile the code, use this command
```sh
$ javac main.java
```

Run the code
```sh
$ java main
```
